Generally Recognized As Safe Open Source Software

The MITRE Corporation is seeking to curate a list of Open Source Software that is in wide deployment and use, and could therefore be reasonably "Generally Recognized As Safe" for general use. If you would like to add to the list, submit an issue or (better yet) a pull request with the appropriate changes.

Projects

  • Apache HTTPD Server Hypertext Transfer Protocol (HTTP) server
  • Linux Kernel UNIX-like kernel behind the Linux operating system
